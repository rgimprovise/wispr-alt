# Беловик iOS

Кастомная iOS-клавиатура «Беловик» с кнопкой диктовки + основное приложение для записи и распознавания. Архитектура та же что в брифе:

- **Main app** (SwiftUI, light Belovik-стиль) — onboarding + recording UI с полноценным доступом к микрофону
- **Keyboard extension** (UIKit, паперно-графитовая палитра) — QWERTY/ЙЦУКЕН + кнопка 🎤
- **App Group** для обмена транскриптом

## ⚠️ Про AirDrop и установку на чужой iPhone

**AirDrop`ом установить на другой iPhone не получится** без Apple Developer Program ($99/год). iOS отказывается ставить .ipa чей provisioning profile не включает UDID целевого устройства.

С **бесплатным Apple ID** возможен **только один путь**: ваш Mac → ваш iPhone по кабелю через Xcode. Билд работает 7 дней, потом нужно пересобрать. Партнёру передать **нельзя** до покупки Developer Program.

Если когда-нибудь купите Developer Program → пакетим `.ipa`, AirDrop`ом или TestFlight'ом раздаём.

## Установка на свой iPhone (один раз ~10 мин)

### Требования

- macOS 14+ с **Xcode 15.4+** (App Store)
- iPhone с **iOS 16+**, кабель Lightning/USB-C
- На iPhone включить Developer Mode: **Settings → Privacy & Security → Developer Mode → On**, перезапустить телефон
- Apple ID (любой бесплатный — для подписи)

### Шаг 1. Сгенерировать Xcode-проект

```bash
brew install xcodegen
cd ~/Desktop/wispr-alt/ios
xcodegen generate
open Belovik.xcodeproj
```

### Шаг 2. Подписать оба target'а

В Xcode для **обоих** target'ов (`Belovik` и `BelovikKeyboard`):

1. В левом сайдбаре кликните по проекту `Belovik` (синяя иконка сверху)
2. По центру выберите target → вкладка **Signing & Capabilities**
3. **Team:** выберите ваш Apple ID. Если в списке нет — Xcode → Settings → Accounts → + Apple ID
4. Bundle Identifier: должен быть **уникальным во всем мире**. Префикс `com.wispralt.app` может быть занят. Переименуйте в свой обратный домен, например `com.<вашлогин>.belovik` (для главного target'а) и `com.<вашлогин>.belovik.keyboard` (для keyboard target'а)

### Шаг 3. App Group (обязательно)

Для обмена транскриптом между приложением и клавиатурой:

1. На каждом target'е — **+ Capability** → **App Groups**
2. Нажать **+** под списком App Groups → создать `group.com.<вашлогин>.belovik`
3. **Включить тоггл** на этой группе для обоих target'ов

После этого откройте `ios/wispr-alt/SharedStorage.swift` И `ios/wispr-alt-keyboard/SharedStorage.swift`, замените:

```swift
static let appGroup = "group.com.wispralt.app"
```

на ваше значение `group.com.<вашлогин>.belovik`. Сохраните, билдите.

### Шаг 4. Сборка и установка

1. Подключите iPhone по USB. На iPhone подтвердите «Trust this computer»
2. В верхнем тулбаре Xcode выберите подключённый iPhone (вместо Simulator)
3. Нажмите ▶ (Cmd+R) → Xcode компилирует и копирует приложение на iPhone
4. На iPhone в первый раз: **Settings → General → VPN & Device Management → ваш Apple ID → Trust**
5. Откройте приложение

### Шаг 5. Включить клавиатуру

В приложении:
1. **«1. Разрешить микрофон»** → Allow
2. **«2. Включить клавиатуру»** → откроется Settings, путь:  
   *Settings → General → Keyboard → Keyboards → Add New Keyboard… → Беловик*
3. **«3. Включить Allow Full Access»** → в том же экране тапните по «Беловик» → включите тоггл

### Шаг 6. Использовать

В Notes / Telegram / Mail / любом приложении с текстовым полем:

1. Тап в поле → стандартная клавиатура поднимается
2. Удержать **🌐** (globe) → выбрать **Беловик**
3. Тап **🎤** на клавиатуре Беловик → откроется наше приложение
4. Скажите фразу → «Готово» → «Передать в клавиатуру»
5. Свайп вверх для возврата в исходное приложение
6. Текст вставится автоматически в поле

## Перезапись после 7 дней

Free-tier signing → билд истекает через 7 дней. На iPhone приложение перестанет открываться. Чтобы продлить — снова `Cmd+R` в Xcode (с подключённым iPhone). Установка переписывает существующую копию, ничего не теряется.

## Архитектурные решения

- **Запись в основном приложении, не в keyboard extension:** Apple ограничивает аудио в keyboard extensions, и приложения такой архитектуры регулярно реджектят в App Store. Поэтому keyboard просто триггерит deep link.
- **Передача транскрипта через App Group UserDefaults:** легковесно, без сложной IPC, синхронно.
- **Polling в `viewWillAppear`:** keyboard проверяет shared storage каждый раз когда становится видимой → если есть pending → вставляет.
- **Нет красного:** микрофонная клавиша графитовая, не красная — следуем брендбуку Belovik.

## Известные ограничения MVP

- Без автокоррекции, emoji-клавиатуры, цифровой раскладки 123
- Цифры можно ввести через стандартную iOS-клавиатуру (тап 🌐 → выбрать стандартную)
- Backend URL зашит, не меняется из UI
- Дизайн минималистичный — упор на функциональность
- iOS не позволяет программно вернуться в исходное приложение → нужно вручную свайпнуть. Это ограничение iOS, не баг.
