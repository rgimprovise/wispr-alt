# А-ГОЛОС — инструкция для тестировщика

Привет! Это приложение для **голосового ввода текста**. Принцип простой: вы
нажимаете горячую клавишу, говорите вслух — и через пару секунд продиктованный
текст появляется в том приложении, где у вас стоял курсор (Telegram, заметки,
Word, почта — что угодно).

Ваша задача — **попробовать его в реальной работе** и записать всё, что покажется
непонятным, неудобным или сломанным. Не нужно «искать ошибки» специально —
просто пользуйтесь приложением как обычно, а в этот документ записывайте всё
странное.

## Что вам понадобится

- Mac, Windows-компьютер или Android-телефон.
- Микрофон (встроенный — нормально, наушники с микрофоном — лучше).
- Интернет — приложение отправляет вашу речь на сервер для распознавания.
- Адрес электронной почты, на который мы пришлём код для входа.
- Тихое место для диктовки. Шумное кафе тоже подойдёт — это часть теста.

## Установка

### macOS

1. Скачайте файл `AGOLOS_2.0.0-4_x64.dmg` (или с похожим номером — главное, чтобы
   расширение было `.dmg`) из ссылки, которую вам прислали.
2. Откройте этот файл двойным кликом — появится окошко с иконкой приложения.
3. Перетащите иконку в папку Applications (она там же справа).
4. Откройте папку Applications, найдите **А-ГОЛОС** (или AGOLOS), запустите.
5. **Скорее всего, появится предупреждение**: *«Приложение от неизвестного
   разработчика, нельзя открыть»*. Это нормально, мы пока без подписи Apple.
   Чтобы обойти:
   - Откройте **Системные настройки → Конфиденциальность и безопасность**.
   - Прокрутите вниз — там будет надпись *«А-ГОЛОС было заблокировано»* и
     кнопка **«Открыть всё равно»**. Нажмите её.
6. Запустите приложение ещё раз — теперь должно открыться.

> ⚠️ **Если у вас уже стоит старая версия** (например, под названием «Беловик»
> или «wispr-alt») — **сначала удалите её**: правый клик по иконке в Applications
> → «Переместить в корзину». Иначе системе может «запомниться» старое название,
> и в Dock будет показываться неправильное имя.

### Windows

1. Скачайте файл `AGOLOS_2.0.0-4_x64.msi`.
2. Двойной клик — откроется установщик. Жмите «Далее» / «Установить».
3. **Скорее всего, Windows покажет SmartScreen**: *«Защита Microsoft Defender
   предотвратила запуск нераспознанного приложения»*. Нажмите «Подробнее» →
   «Выполнить в любом случае».
4. После установки приложение появится в меню «Пуск» под именем **А-ГОЛОС**.

### Android

1. Скачайте файл `app-release.apk`.
2. Откройте его на телефоне. Возможно, телефон попросит разрешить установку из
   неизвестных источников — разрешите.
3. Установите. Приложение появится в списке программ под именем **А-ГОЛОС**.

## Первый запуск и вход

При первом запуске приложение попросит ввести email и пройти простую авторизацию.

1. Введите свой email. Нажмите «Продолжить».
2. Если этот email **в первый раз** — на почту придёт письмо с кодом из 6 цифр.
   Введите код. Готово, вы зашли.
3. Если у вас уже есть **пароль** для этого email — введите его. Если забыли —
   нажмите «Войти по коду» и получите код на почту.
4. После входа приложение предложит **установить пароль** (необязательно, но
   удобно — чтобы в следующий раз не ждать письмо).

### Что должно произойти, что записать

✅ Письмо с кодом приходит за 5–30 секунд. Если не пришло — **проверьте папку
«Спам»**, и запишите этот факт в отчёт.

✅ Код из 6 цифр работает с первого раза. Если приходит ошибка — запишите.

✅ После входа экран меняется — появляется кнопка для записи или подсказка про
горячую клавишу.

## Что вы увидите на экране

### Главное окно

Это «настройки». Здесь можно:
- увидеть, какой клавишей запускается запись (по умолчанию **F5** на десктопе);
- сменить эту клавишу на любую другую;
- выбрать **стиль** обработки текста (см. ниже);
- проверить, что приложение вообще работает (кнопка «Тестовая вставка»);
- выйти из аккаунта.

### Овeрлей (Overlay)

Это маленькая «таблетка», которая появляется **поверх любого приложения**,
когда вы записываете. На ней показывается:
- цветная точка слева — индикатор состояния;
- надпись «Запись» / «Расшифровка» / «Готов»;
- таймер записи;
- ваш текст по мере распознавания (см. сценарий 4 ниже).

### Стили обработки

В настройках есть выпадающий список стилей. Выбирайте под ситуацию:

| Стиль | Когда использовать |
|---|---|
| **Чистка** | Просто убрать «эээ», «ну», расставить точки — без правок текста. |
| **Деловой** | Письмо коллеге, отчёт. Формальный тон. |
| **Неформальный** | Сообщение другу, чат. Живой разговорный стиль. |
| **Краткий** | Длинная мысль → 2-3 предложения сути. |
| **Telegram-пост** | Пост в канал/чат: с заголовком, абзацами. |
| **Email** | Письмо: «Здравствуйте, …», абзацы, подпись. |
| **Задача** | Запись задачи: заголовок, контекст, список «что сделать». |

## Сценарии для проверки

Пройдите по этим сценариям по порядку. После каждого — отмечайте у себя
«работает» / «не работает» / «странно» с пояснением.

### Сценарий 1. Короткая диктовка в заметки

**Цель:** убедиться, что базовый поток вообще работает.

1. Откройте любое приложение для заметок (на Mac — «Заметки», на Windows —
   «Блокнот», на телефоне — «Заметки» / «Keep»).
2. Поставьте курсор в пустое поле.
3. Нажмите **F5** (на десктопе) или кнопку записи (на телефоне). Должна
   появиться «таблетка» овeрлея с надписью «Запись».
4. Произнесите вслух: **«Привет, это первая запись для теста»**.
5. Нажмите **F5** ещё раз (или отпустите кнопку записи на телефоне).
6. В овeрлее появится «Расшифровка», потом он закроется.
7. **В вашу заметку должен вставиться текст**: *«Привет, это первая запись
   для теста.»* (с заглавной буквы и точкой в конце).

**Что записать:**
- ✅ / ❌ Текст появился в заметке.
- Сколько секунд прошло между «отпустил клавишу» и «текст появился»?
- Текст совпадает с тем, что вы сказали? Если есть ошибки — какие?

---

### Сценарий 2. Длинная диктовка (важно!)

**Цель:** проверить, что приложение не «зависает» на длинных записях. Раньше
была проблема: диктовка дольше минуты — и приложение «застревало» в стадии
«Расшифровка», текст никогда не появлялся.

1. Откройте заметки.
2. Нажмите F5.
3. **Говорите без остановки минимум 2 минуты**. Не нужно ничего сложного —
   расскажите, что у вас за день, что планируете на завтра, любые мысли.
   Главное — не молчите.
4. Нажмите F5 для остановки.
5. **Ждите.** Это нормально, что текст появится не сразу — серверу нужно
   время. Допустимо до 60 секунд ожидания.
6. Текст должен вставиться в заметки — большой кусок, как вы и говорили.

**Что записать:**
- ✅ / ❌ Текст появился (НЕ застрял в «Расшифровка» навсегда).
- Сколько секунд от «отпустил клавишу» до «текст в заметках»?
- Текст вообще похож на то, что вы говорили?
- В овeрлее во время записи появляется **ваш текст по мере того, как вы
  говорите** (live-превью)? Или экран пустой?
- **Цветная точка слева в овeрлее каждую секунду слегка увеличивается?**
  (Это маленький «пульс», диагностический индикатор. Если не пульсирует —
  обязательно запишите.)

---

### Сценарий 3. Все 7 стилей по очереди

**Цель:** проверить, что разные стили действительно по-разному форматируют
текст.

Для каждого из 7 стилей:

1. В настройках главного окна выберите стиль.
2. Откройте заметки.
3. Продиктуйте **одну и ту же фразу для всех стилей**, например:

   > *«Привет, я вчера созвонился с командой, обсудили план на следующий
   > месяц, нам нужно сосредоточиться на трёх вещах: доделать авторизацию,
   > починить баги на айфоне и запустить рекламную кампанию которую мы уже
   > два месяца откладываем. Срок две недели.»*

4. Запишите, как текст отформатирован (или сделайте скриншот).

**На что смотреть:**

| Стиль | Как должен выглядеть результат |
|---|---|
| **Чистка** | Тот же текст почти без правок, но с пунктуацией и абзацами. |
| **Деловой** | Формальный тон, «Я» вместо «короче я», абзацы по темам. |
| **Неформальный** | Похож на живой текст, «короче» сохранено, но без «эээ». |
| **Краткий** | Короче в 2–3 раза, маркированный список из 3 пунктов. |
| **Telegram** | Заголовок-крючок, потом 2–3 коротких абзаца, может быть список. |
| **Email** | Начинается с «Здравствуйте», потом текст, абзацы. |
| **Задача** | Заголовок (например, «Доделать авторизацию и починить баги»), потом блоки «Контекст», «Что сделать» (список), «Срок». |

**Что записать:**
- Каждый стиль выглядит по-разному, как описано выше? Или всё похоже на
  «один и тот же текст без структуры»?
- В тексте видны странные значки **`#`**, **`-`**, **`*`**, **`**`**? Их
  быть не должно — если вы их видите, обязательно запишите.
- Какие стили показались вам **полезными**, а какие — **не отличаются от
  обычного текста**?

### Сценарий 4. Live-превью в овeрлее

**Цель:** во время записи в овeрлее должен **появляться текст по мере того,
как вы говорите** — приблизительный, может с ошибками, но он там есть.

1. Откройте заметки.
2. Нажмите F5.
3. Говорите медленно: **«Раз. Два. Три. Четыре. Пять.»**, делая паузы
   между числами.
4. **Смотрите на овeрлей**, а не в заметки.

**Что должно происходить:**
- Через 2–3 секунды после первого «раз» в овeрлее появляется текст «Раз».
- Через секунду добавляется «Два». И так далее.
- Текст обновляется примерно раз в секунду.
- Цветная точка слева в овeрлее **пульсирует** (слегка увеличивается каждую
  секунду — даже когда вы молчите).

**Что записать:**
- ✅ / ❌ Текст появляется в овeрлее во время записи.
- Через сколько секунд после начала речи появилось первое слово?
- Точка пульсирует? Только во время записи или всегда?

### Сценарий 5. Голосовые команды «Логос»

**Цель:** в любую диктовку можно встроить команды, которые меняют текст
прямо во время записи. Команда начинается со слова **«Логос»**.

Попробуйте каждую из этих диктовок и проверьте, что слово «Логос» **не
попадает в финальный текст** — оно должно отработать как команда:

1. **«Сегодня встречался с командой обсудили планы на квартал. Логос, удали
   последнее. Завтра созвонимся с клиентом.»**
   Ожидание: фраза «обсудили планы на квартал» исчезает. Остаётся:
   *«Сегодня встречался с командой. Завтра созвонимся с клиентом.»*

2. **«Сегодня нужно доделать отчёт. Логос, новый абзац. И отправить его в
   офис.»**
   Ожидание: получается два абзаца, разделённых пустой строкой.

3. **«Привет коллеги. Логос, поставь восклицательный знак. Сегодня обсуждаем
   план.»**
   Ожидание: *«Привет, коллеги! Сегодня обсуждаем план.»*

4. **«Встреча с Иваном в три часа. Логос, замени Иван на Иван Петрович.»**
   Ожидание: *«Встреча с Иваном Петровичем в три часа.»*

**Что записать:**
- Какие команды сработали, какие — нет.
- Слово «Логос» само попало в финальный текст хоть раз? (Это баг.)

### Сценарий 6. Вставка в разные приложения

**Цель:** убедиться, что текст вставляется не только в заметки, но и в
любое приложение.

Попробуйте надиктовать одно короткое предложение в каждое из:

- Telegram (поле ввода в чате).
- Браузер: поле поиска Google.
- Word / Google Docs (если есть).
- Адресная строка браузера.
- Почта (тело письма в Gmail / Apple Mail / Outlook).

**Что записать:**
- В каких приложениях текст вставился, а в каких — нет.
- Если не вставился — запишите, как именно «не вставился»: совсем ничего,
  или текст ушёл в какое-то другое поле, или вставилось не то.

### Сценарий 7. Смена горячей клавиши (только десктоп)

1. В главном окне приложения найдите раздел «Горячая клавиша».
2. Поменяйте её на любую другую — например, **F8** или **Cmd+Shift+G** на Mac.
3. Закройте главное окно (приложение не выключайте).
4. Откройте заметки. Нажмите **новую** клавишу.

**Что записать:**
- ✅ / ❌ Запись началась по новой клавише.
- ✅ / ❌ Старая клавиша (F5) **больше не работает**.

### Сценарий 8. Запись в шумной обстановке

**Цель:** проверить как приложение справляется с реальностью.

Включите дома фоновый шум: телевизор, вентилятор, музыку (негромко).
Продиктуйте одну фразу: **«Завтра нужно встретиться с командой и обсудить
план на следующую неделю.»**

**Что записать:**
- Текст распознался? С ошибками? Какими?

### Сценарий 9. Что происходит, если интернет пропал

1. Включите режим «В самолёте» / отключите Wi-Fi.
2. Попробуйте надиктовать что-нибудь.

**Ожидание:** приложение должно показать понятную ошибку («нет сети»,
«сервер недоступен»), а не зависнуть навсегда.

**Что записать:**
- Какое именно сообщение об ошибке вы увидели?
- Через сколько секунд оно появилось?
- После того как вернули интернет — приложение работает дальше как ни в
  чём не бывало, или нужно перезапустить?

## Как сообщить о проблеме

Если что-то сломалось или показалось странным — напишите нам. Самый удобный
формат отчёта:

```
Что я делал:    [например: записывал длинную диктовку 2 минуты]
Что я ожидал:   [например: текст должен был вставиться в Telegram]
Что произошло:  [например: овeрлей застрял в "Расшифровка" на минуту, потом
                 сам закрылся, а в Telegram ничего не вставилось]

Платформа:      [Mac / Windows / Android]
Версия:         [её можно посмотреть в главном окне приложения, например 2.0.0-4]

Скриншот / видео: [если есть]
```

**Что особенно полезно прислать:**
- **Скриншот овeрлея** в момент, когда что-то идёт не так.
- **Видеозапись экрана** (на Mac — Cmd+Shift+5, на Windows — Win+G), если
  баг сложно описать словами.
- **Буквальный текст**, который вы сказали, и буквальный текст, который
  вставился (можно прислать оба варианта, мы сравним).

## Если что-то совсем не работает

### Mac: вместо «А-ГОЛОС» в Dock показывается «Беловик» или «wispr-alt»

Это значит, что у вас стояла старая версия и macOS запомнил её имя.

1. Откройте Applications, найдите старую копию (она может называться
   «Беловик» или «wispr-alt» или «AGOLOS»).
2. **Удалите все** копии в корзину.
3. Очистите корзину.
4. Перезапустите Mac (это важно — иначе кэш имён не сбросится).
5. Установите свежий DMG как описано в начале.

### Mac: при нажатии F5 ничего не происходит

Скорее всего, не хватает разрешения «Универсальный доступ» (Accessibility).

1. **Системные настройки → Конфиденциальность и безопасность → Универсальный
   доступ**.
2. Найдите в списке **А-ГОЛОС**. Включите переключатель.
3. Если приложения нет в списке — нажмите **+** и добавьте его из Applications.
4. Перезапустите А-ГОЛОС.

### Mac: текст «расшифровывается», но в заметки ничего не вставляется

Тоже про разрешения, но другое — **«Универсальный доступ»** (см. выше) +
дополнительно **разрешение на использование микрофона**:

1. Системные настройки → Конфиденциальность и безопасность → **Микрофон**.
2. Найдите А-ГОЛОС, включите.

### Windows: SmartScreen не даёт запустить установщик

Нажмите **«Подробнее»** в окне предупреждения, потом **«Выполнить в любом
случае»**. Это разовая операция.

### Android: овeрлей не появляется поверх других приложений

Нужно разрешение на показ поверх других программ + разрешение «Специальные
возможности»:

1. Настройки → Приложения → А-ГОЛОС → **Поверх других приложений** → Разрешить.
2. Настройки → Специальные возможности → А-ГОЛОС → Включить.

### Не приходит письмо с кодом

1. Подождите 1–2 минуты — иногда задержка.
2. Проверьте папку **«Спам»**.
3. Проверьте, что вы ввели email **без опечаток**.
4. Если уже больше 5 минут и кода нет — напишите нам, это баг.

вы опишете, что увидели, тем быстрее мы это починим. Не стесняйтесь писать

пользователя ценнее любого теста.
