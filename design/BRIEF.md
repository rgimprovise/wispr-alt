# А-ГОЛОС — design brief

The current brand and design system live in **[../brand/BRAND.md](../brand/BRAND.md)**
— that's the single source of truth: identity, color tokens, typography,
radii, shadows, voice/copy rules, logo construction, и список don'ts.

Application of the design system is reflected in:

- `app/src/tokens.css` — desktop CSS variables
- `app/src/styles.css`, `app/src/overlay.css` — desktop UI
- `android/app/src/main/res/values/colors.xml`, `values-night/colors.xml` — Android
- `ios/wispr-alt/AgolosTokens.swift` — iOS
